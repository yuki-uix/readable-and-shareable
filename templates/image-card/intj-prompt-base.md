# Prompt Base — INTJ Style
# 认知偏好：框架先行 · 演绎逻辑 · 精确语义 · 强收口
# 用法：替换 {{}} 占位符，GLOBAL RULES 保持不变

A clean systems-diagram infographic, portrait 2:3 ratio.
Style: technical-document illustration — precise, geometric, hierarchical. NOT sketchnote, NOT hand-drawn.
Background: very light gray-white #F7F7F5. Primary text: near-black #1C1C1A. Secondary text: #6B6A64.
Accent color: cold blue #2563EB — used ONLY on primary framework nodes and conclusion bar. No other colors.

=== PAGE PROPORTION RULES ===
- Header: top 12% — title only, one line
- Framework zone: 25% — the core model, most visually prominent element on page
- Supporting detail zones: 45% — 2–3 secondary sections, visually smaller than framework
- Conclusion bar: 10% — full-width, blue left border
- Footer: 8%

=== VISUAL RULES ===
- Lines: clean, precise, 1px, near-black — NO hand-drawn texture, NO wobble
- Boxes: 4–6px corner radius, 0.5px border, geometric — NOT organic or tilted
- Primary nodes: blue #2563EB border or label — larger size than secondary nodes
- Secondary nodes: gray #EBEBEA fill, near-black border — visually smaller
- Arrows: thin, labeled with logical operator — NO unlabeled arrows
- Typography: structured print-feel, NOT handwriting — clear weight contrast 600/400

=== HEADER (top 12%) ===
One line, weight 600, near-black, 20px:
  {{TITLE}}

=== FRAMEWORK ZONE (most prominent, top of content area) ===
Show the core model / system structure first.
{{FRAMEWORK_DESCRIPTION}}

Primary nodes use blue #2563EB labels.
Logical relationships shown with labeled arrows.
This zone answers: "What is the system?"

=== SUPPORTING ZONES (below framework, visually subordinate) ===
{{SUPPORTING_ZONES}}

Each supporting zone:
- Gray fill #EBEBEA, near-black border
- Near-black label, weight 600, smaller than framework labels
- Max 2 bullet points, 12px secondary text
- Connected to framework with thin labeled arrow

=== CONCLUSION BAR (full width, bottom of content) ===
Full-width block, 4px solid blue left border, light gray fill #EBEBEA:
  {{CONCLUSION — one precise sentence, weight 600, near-black, underlined}}

=== FOOTER ===
12px, secondary gray text, right-aligned:
  via readable-and-shareable

=== GLOBAL RULES ===
- Background: #F7F7F5 everywhere — no white, no cream, no color zone fills
- Cold blue #2563EB: ONLY on primary framework nodes and conclusion bar border
- Gray #EBEBEA: secondary node fills and conclusion bar background
- Near-black #1C1C1A: all text, borders, arrows
- NO hand-drawn texture, NO organic shapes, NO sketchnote aesthetic
- NO decorative icons, NO illustrations, NO stars, NO doodles
- Arrows: 1px thin line, small arrowhead, logical label on line (「因此」「约束」「导致」「≠」)
- Visual hierarchy enforced by SIZE: framework nodes larger, detail nodes smaller
- Deductive reading order: framework → details → conclusion (top to bottom)
- Conclusion bar is the only full-width element in content area
- All Chinese text: clear, correct, complete, no garbled characters
