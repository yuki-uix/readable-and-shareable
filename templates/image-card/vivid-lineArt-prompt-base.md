# Prompt Base — vivid-lineArt Style
# 用法：把 === CONTENT === 部分替换为具体内容，其余 GLOBAL RULES 保持不变

A hand-drawn educational infographic, portrait 2:3 ratio.
Style: teal-and-black line-art sketchnote, inspired by "The Six Vivid Quick Tricks" visual grammar.
Background: pure white #FFFFFF. Text: near-black #1A1A1A.
Accent color: ONE teal only — #00B5B5. NO other colors anywhere.

=== VIVID LINEĀRT STYLE RULES ===
- Background: pure white, NO cream, NO color fills behind text sections
- Boxes: hand-drawn rectangular border in teal #00B5B5, white interior, slightly tilted/organic edges
- Arrows: hand-drawn curvy black arrows; write the action verb ON the arrow line (e.g. "leads to →")
- Typography: large mixed-case handwritten title; body text small neat handwriting; key terms underlined
- Sequence markers: black circled numbers ① ② ③ placed at page margins pointing inward — NOT inside grid
- Icons: simple 2-3 line drawings inside boxes (functional, not decorative)
- NO background color blobs, NO stars, NO macaron fills, NO colored zone backgrounds
- Strictly two-color: teal + black only

- Black circled number ① ② ③...

=== GLOBAL RULES ===
- STRICTLY two colors: teal #00B5B5 + near-black #1A1A1A only
- Pure white background everywhere — no fills, no blobs, no cream
- All box borders: hand-drawn teal, slightly tilted, never perfect rectangles
- All arrows: hand-drawn black curves, verb label ON arrow line
- All sequence numbers: black circle outline with number inside, at PAGE MARGINS
- NO decorative elements (no stars, no clouds, no background shapes)
- Icons inside boxes: ultra-simple 2-3 line drawings only
- Key terms in body text: underlined (straight underline, black)
- Font size hierarchy: title LARGE > box labels MEDIUM > body text SMALL
- All Chinese text: clear, correct, complete, no garbled characters
