# 图片卡片模板 — 截图操作指引

生成的图片卡片是一套 1:1 正方形卡片，适合直接截图后发布到微信、小红书、X 等社交平台。

---

## 使用流程

### 1. 生成 HTML

在 Claude Code 中运行 `/viz`，选择"图片卡片"，按照提示确认提取的核心观点后，Claude 会生成填充完毕的单文件 HTML。

### 2. 在 Chrome 中打开

用 Chrome 打开生成的 HTML 文件（直接拖入浏览器即可）。

### 3. 调整窗口宽度

将 Chrome 窗口宽度调整为约 **600px**，确保每张卡片完整显示为正方形。

> 快速调整方法：打开 DevTools（`⌘+Option+I`）→ 切换设备模式（`⌘+Shift+M`）→ 手动输入宽度 600。

### 4. 截图

```
⌘+Shift+P  →  输入 "Capture screenshot"  →  回车
```

Chrome 会自动下载当前可见区域的截图。

### 5. 逐张截图

每张卡片各截一次。滚动到下一张卡片后重复步骤 4。

---

## 注意事项

- **浏览器版本**：Chrome 88+ 才支持 `aspect-ratio`，请确保已更新
- **系统缩放比**：建议设为 100%（系统偏好设置 → 显示器 → 缩放），避免不同设备截图尺寸不一致
- **深色模式**：卡片配色固定为浅色，不随系统主题变化，截图结果始终一致
- **字体**：使用系统默认中文字体（PingFang SC / Microsoft YaHei），确保文字清晰
