Isolate + Write：任务隔离与外置记忆
— 模型分级 + Session 跨轮延续
核心框架
两种隔离：上下文隔离（Isolate）+ 时间隔离（Write）
Isolate 把大任务拆成子任务，在独立上下文窗口执行，完成后只把结论交回主 Agent。Write 把不需要时刻在场的信息搬到文件系统，context window 是 RAM，文件系统才是 Disk。
Isolate：模型分级策略（成本差 15×）
