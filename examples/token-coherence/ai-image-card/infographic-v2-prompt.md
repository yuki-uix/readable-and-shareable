=== GLOBAL RULES ===
- Use ONLY macaron palette — all at same softness, none more saturated than others
- All borders: hand-drawn wobble, organic, never perfect geometric shapes
- Hand-painted color fill gap at all edges
- Gold ✦ stars between zones as dividers
- Zone 3 is FULL WIDTH and visually most prominent — largest zone on page
- NO English text in small body elements — Chinese labels only (token / IA acceptable as loanwords)
- Cute hand-drawn doodle icons: lightbulb, speech bubble, warning triangle, star
- EXTREME font size contrast: section titles large, body text small — hierarchy by size not color
- Key conclusion sentences always in hand-drawn rectangular boxes
- Zone 5 uses RADIAL layout (not Venn circles) — center oval with 3 arrows radiating out
- Compact — minimize empty whitespace
- All Chinese text: clear, correct, complete, no garbled characters
