# ENFP Style — 意义星图

**认知偏好**：可能性爆发 · 价值驱动 · 人际联结 · 意义寻找

**信息组织方式**：星座聚焦式——从多个散落的意义闪光点出发，用「这些为什么重要」的线索把它们连接成一个有意义的星座。

---

## 核心美学

温暖的意义星图。像一张画在日记里的「这些事情为什么对我重要」的连接图——不是论证，不是步骤，而是把散落的「闪光点」连接成一个有意义的星座。

比 ENTP 更暖、更柔、更「关于人」；比 INFP 更外向、更多连接节点、更有能量感。

---

## 设计 Token

```
--bg:        #FEFCF7   /* 极暖白 */
--surface:   #FFF5E6   /* 浅杏，温暖 */
--text:      #2A1F14   /* 深暖棕 */
--text2:     #8A7060
--border:    rgba(180,100,40,0.10)
--star:      #E8832A   /* 暖橙，核心意义节点 — 比 ENTP 的橙更柔 */
--star-bg:   #FFF0DC
--link-a:    #D4845A   /* 珊瑚，人际/连接类节点 */
--link-b:    #7AB87A   /* 柔绿，价值/意义类节点 */
--link-c:    #8B78C8   /* 柔紫，想象/可能性类节点 */
```

颜色原则：**3 色星座系统**。
- 暖橙：核心意义节点（最重要的「为什么」）
- 珊瑚：人与人的连接节点
- 柔绿：价值观/意义锚点
- 柔紫：可能性/想象力节点

4 色都是柔和色，不刺眼。放在一起是「星座」感，不是「数据图表」感。

---

## 布局原则

**星座式散点网络**，中心有意义锚点：

```
  [可能性节点]    [人际节点]
        \           /
    [柔紫]       [珊瑚]
          \     /
        [核心意义]  ← 暖橙，最大
          /     \
    [柔绿]       [柔绿]
        /           \
  [价值节点]    [价值节点]
```

- 中心最大节点：核心「为什么」（暖橙）
- 围绕中心的节点：3-5 个，颜色按类型区分
- 连线：柔和曲线，不加箭头（联结，不是流程）
- 允许节点之间有多条连线（网络感，不是树状）
- 散落的「闪光词」：几个无连线的短语飘在空白处

---

## 排版

- 节点文字：短语为主，情感词汇优先
- 连线：细，柔和，无标签
- 闪光词：斜体，小号，像灵感碎片
- 标题：温暖叙事感，第一人称或第二人称
- 结语：「意义感」收口，不是结论

---

## 特有视觉元素

- **3 色节点系统**：珊瑚/柔绿/柔紫，按语义分类
- **核心意义节点**：暖橙，最大，有轻微光晕
- **无标签连线**：纯连接，不解释
- **闪光词**：散落在空白处的短语片段，像便利贴
- **星座感**：整体像一个星座图，而不是流程图

---

## 收口方式

**意义确认**，温暖但不煽情：

- 「这些事情为什么重要——因为它们都在说同一件事」
- 「也许 [核心意义] 一直在那里，等你命名它」

ENFP 读者不需要结论，他们需要**感受到这张图帮他们把分散的感受连成了一个整体**。

---

## 不适合

- 冷色调（失去温度感）
- 强逻辑推导（那是 ENTP——虽然同样发散，但驱动力不同）
- 步骤化（那是 ISFJ）
- 单一颜色（ENFP 需要多色来表达多维度的意义）

### 签名视觉元素（v2）— 必须实现的布局语法

**① 星点卡（Spark Card）**
- 左侧橙色圆点（8px，`--star`），紧跟意义词（14px bold），下方一行说明（11px `--text2`）
- 每张卡片 `--star-bg` 背景，`0.5px --border` 边框，`border-radius: 6px`，`padding: 10px 14px`
- 3-5 张，垂直排列，不加序号

**② 三色标签系统（Category Tags）**
- 珊瑚色（`--link-a`）= 人际/连接类 概念
- 柔绿（`--link-b`）= 价值/意义类 概念
- 柔紫（`--link-c`）= 想象/可能性类 概念
- 每个标签：`border-radius: 10px`，`padding: 2px 8px`，`font-size: 10px`，放在卡片右上角

**③ 连接叙事行（Why It Matters）**
- 星点卡之间加一行浅橙虚线分隔 + 斜体连接句（11px `--text2`，如「这让我看见……」）
- 不是所有卡片都需要，2-3 处即可，用于打通意义线索

**④ 开放收口（Open Star）**
- 末尾一个虚线边框星点（`border: 1.5px dashed var(--border)`），内容为「还在寻找……」
- 表示星座尚未完整，保留探索空间

## 与其他风格对比

| | ENFP | ENTP | INFP |
|---|---|---|---|
| 中心驱动 | 意义/价值（Fi）| 论点/逻辑（Ti）| 感受/叙事（Fi）|
| 节点颜色 | 3 色（珊瑚/绿/紫）| 单色（烈橙中心）| 4 色情绪区块 |
| 连线 | 无标签柔和曲线 | 带方向的有机箭头 | 无，区块自说明 |
| 收口 | 意义确认 | 开放反问 | 开放邀请 |
| 工具 | AI 生成 | AI 生成 | AI 生成 |
