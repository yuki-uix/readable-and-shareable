# ESFP Style — 体验海报

**认知偏好**：体验优先 · 即兴活力 · 感官愉悦 · 高能量

**信息组织方式**：体验海报式——把内容压缩成最高能量的视觉冲击，用大字、强色、即时感传递核心，不讲道理只给感受。

---

## 核心美学

高能量的体验海报。像一张音乐节海报或一张活动邀请卡——色彩饱和、充满活力、让人想参与。
不解释，不分析，直接给你一种「这个很好玩 / 很值得」的感受。

---

## 设计 Token

```
--bg:        #FFFDF7   /* 暖白 */
--burst-a:   #FF6B35   /* 活力橙 */
--burst-b:   #FFB800   /* 阳光黄 */
--burst-c:   #4ECDC4   /* 活力青 */
--burst-d:   #FF6B9D   /* 亮粉 */
--burst-e:   #7BC67E   /* 鲜绿 */
--text:      #1A1A18   /* 近黑 */
--text2:     #6A6A60   /* 次要 */
```

颜色原则：**高饱和多色**，像彩虹。颜色是能量，不是语义。
所有颜色一起出现要有活力感，不是混乱——靠字重和留白控制。

---

## 布局原则

**体验标题 + 感受列表 + 邀请**，像活动宣传单：

```
[超大标题 — 感受是什么]

✦ 这件事让你感觉……
✦ 你会想到……
✦ 最好的部分是……

[邀请语 — 来体验吧]
```

- 标题最大，高饱和色块底
- 感受列表：`✦` 符号，每条一种颜色
- 大量彩色装饰元素：圆点、色块、几何形状
- 邀请语：温暖，直接，第二人称

---

## 特有视觉元素

- **超大标题色块**：整行彩色底色
- **彩色 ✦ 列表**：每条感受一种颜色
- **装饰圆点/星形**：散落在空白处，增加活力感
- **邀请感收口**：「你也来感受一下」

---

## 收口方式

**邀请体验**，充满活力：

- 「来感受一下——不用想太多，直接试」
- 不分析，不解释，直接邀请

---

## 不适合

- 冷色调（失去活力）
- 高密度信息（压制能量感）
- 逻辑推导（ESFP 要感受，不要分析）
- 黑白灰为主（太沉闷）
