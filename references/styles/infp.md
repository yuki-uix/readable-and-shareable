# INFP Style — 手写的信

**认知偏好**：归纳 · 情感共鸣 · 开放探索 · 叙事弧线

**信息组织方式**：叙事弧线式——从个人感受或困惑出发，经过探索和转折，到达开放式感悟，像一封写给读者的信。

---

## 核心美学

手绘温感信息图。像一封写给读者的信，不是写给自己的报告。
线条有意不整齐，留白多，读者可以把自己的感受填进去。

---

## 设计 Token

```
--bg:        #FFFDF7   /* 暖白，接近手账纸 */
--surface-a: #FFE8EC   /* 玫瑰粉 — 情感/感受区 */
--surface-b: #E8F4E8   /* 苔藓绿 — 观察/事实区 */
--surface-c: #FFF3D6   /* 奶油黄 — 联想/延伸区 */
--surface-d: #E8EEFF   /* 薰衣草蓝 — 疑问/开放区 */
--text:      #2C2416   /* 深暖棕，像钢笔墨水 */
--text2:     #7A6E5F   /* 次要文字 */
--line:      #2C2416   /* 手绘线条，同文字色 */
```

颜色分配原则：每个色块对应一种**情绪功能**，不是视觉装饰。
- 粉 = 主观感受、情绪触点
- 绿 = 客观观察、事实锚点
- 黄 = 自由联想、边界模糊的想法
- 蓝 = 问题、未解决的疑惑

---

## 布局原则

**叙事弧线**，不是流程图：

```
开场感受（故事入口）
  ↓
观察/发现（转折点）
  ↓
联想延伸（意义生长）
  ↓
开放落点（邀请读者）
```

- 垂直单列为主，偶尔左右错位制造节奏感
- 色块不对齐、不等宽——有机排列，不追求网格
- 箭头：弯曲手绘线，不加文字标签，方向暗示逻辑
- 留白：版面填充率不超过 65%

---

## 排版

- 标题：手写感字体（Caveat / Ma Shan Zheng），或普通字体加倾斜
- 正文：小号，行距宽松（1.8），字重轻
- 强调：手绘下划线波浪线，或彩色底色涂抹感
- 数字：圆圈包裹 ①②③，手绘风格
- 结语：不是结论句，是一个问题或邀请语

---

## 视觉元素

- 小插图：简笔人物/植物/物件，1-2 个，点缀用
- 星号 ✦ / 小花 ✿：段落装饰，替代 bullet point
- 波浪线下划线：重点标注
- 色块边缘：轻微不规则，模拟手撕/手剪质感（prompt 描述）

---

## 收口方式

**不用强结论**。结语应该是：
- 一个反问
- 一句邀请（"你有没有……"）
- 一个未完成的句子（"也许……"）

INFP 读者不需要被告知答案，他们需要被**邀请进入对话**。

---

## 不适合

- 演绎逻辑流程（那是 INTJ）
- 高密度信息矩阵（那是 ENTJ）
- 强结论单句收口
- 蓝色/灰色主色调（太冷，失去温度感）
- 完全对称的布局（太机械）

---

## 工具分配

**AI 图像生成**（qwen-image-2.0-pro）

手绘有机质感 = HTML 难以模拟，AI 生成是主路径。
中文字符风险：避免长句，关键词 ≤ 4 字，重要词汇用英文替代测试。

参考 prompt 模板：`templates/image-card/infp-prompt-base.md`

## 与其他风格对比

| | INTJ | INFP | vivid-lineArt |
|---|---|---|---|
| 温度感 | 冷 | 暖 | 中性偏活力 |
| 线条 | 精确几何 | 手绘有机 | 手绘+双色强调 |
| 颜色数 | 1 色（冷蓝）| 4 色（情绪色） | 2 色（青+黑）|
| 收口 | 强结论 | 开放邀请 | 强行动 |
| 工具 | HTML | AI 生成 | AI 生成 |
