# ENFJ Style — 成长路线图

**认知偏好**：他人成长 · 愿景感召 · 整合共识 · 转化叙事

---

## 核心美学

温暖的成长路线图。像一份给某人的「你可以走到这里」的地图——有起点（现在的困境），有转化节点，有清晰的终点（值得去的地方）。
读完感受是「被看见」，然后「被邀请前进」。

---

## 设计 Token

```
--bg:        #FDFAF6   /* 暖白，接近米色 */
--surface:   #F5EFE6   /* 浅琥珀，温暖中性 */
--text:      #2A1F14   /* 深暖棕，像手写字 */
--text2:     #8A7060   /* 次要文字 */
--border:    rgba(120,80,40,0.12)
--amber:     #C97C2A   /* 琥珀橙，核心节点 + 路径线 */
--amber-bg:  #FEF3E2   /* 琥珀浅，背景 tint */
--growth:    #4A7C59   /* 苔藓绿，「到达」节点 + 正向结果 */
--growth-bg: #EEF3EE   /* 绿浅，到达区背景 */
```

颜色原则：**琥珀 = 路径/召唤，绿色 = 到达/成长**。
- 琥珀：路径箭头、核心转化节点、标题强调
- 绿色：终点/结果节点、正向状态
- 两色不混用，形成「旅程感」的视觉方向

---

## 布局原则

**垂直成长路径**，有转化节点：

```
[起点 — 现在的状态/困境]
         ↓ 琥珀路径
[转化节点 A — 看见了什么]
         ↓
[转化节点 B — 发生了什么变化]
         ↓ 路径变宽，颜色渐转绿
[终点 — 可以到达的地方]

旁注：沿途的小感悟，像路标
```

- 起点区：暖灰底，描述「现在」
- 转化节点：琥珀色，描述「看见」和「领悟」
- 终点区：绿色底，描述「可以是什么样子」
- 路径箭头：加粗，有流动感
- 旁注：路径旁的小字，像鼓励的附注

---

## 排版

- 标题：暖棕色，温暖感，不全大写
- 起点/终点：字号略大，给予重量感
- 旁注：斜体，小号，像手写附注
- 收口：一句给读者的话，「你」视角，不是「他们」视角

---

## 特有视觉元素

- **成长箭头**：加粗路径线，从琥珀渐变到绿色
- **转化标签**：小圆形节点，标注「转折点」「看见的时刻」
- **旁注**：路径旁斜排小字，像导师的提示
- **终点光晕**：终点节点有轻微绿色光晕感（subtle box-shadow）
- **「你可以」句式**：收口用第二人称

---

## 收口方式

**邀请前进**，用第二人称：

- 「如果你也在 [起点]，这张图也许是给你的」
- 「[终点状态] 不是天赋，是可以走到的地方」

ENFJ 读者不需要「这是结论」，他们需要**感受到这张图是写给他们的，并且相信自己可以走到终点**。

---

## 不适合

- 冷色调（太理性，失去温度）
- 无方向感的发散网络（那是 ENTP/ENFP）
- 纯逻辑推导（那是 INTP）
- 匿名的「他们」视角（ENFJ 需要直接对话）

## 与其他风格对比

| | ENFJ | INFJ | ISFJ |
|---|---|---|---|
| 叙事视角 | 第二人称「你」 | 整体图景「这个系统」 | 操作步骤「这样做」 |
| 核心色 | 琥珀→绿（成长旅程）| 深紫→深青（深度）| 柔绿（进度完成）|
| 收口 | 邀请前进 | 整体感悟 | 明确完成 ✅ |
| 密度 | 中等，有留白 | 中等，有深度 | 高，步骤化 |
