# ISFP Style — 色彩故事

**认知偏好**：感官美学 · 内在价值 · 安静表达 · 色彩直觉

---

## 核心美学

安静的色彩故事。像一张精心选配的色板——颜色本身就是内容，每种色彩都在说一件事，整体构成一个有意义的「色彩叙事」。
不是信息密集，不是步骤清单，而是通过色彩和留白让读者**感受**到核心概念。

---

## 设计 Token

```
/* ISFP 的色彩是精心选配的，不是语义标注 */
--bg:         #FDFBF7   /* 极暖白，米色调 */
--palette-a:  #7A9E8E   /* 灰蓝绿，克制 */
--palette-b:  #C4856A   /* 暖赭石，温暖 */
--palette-c:  #B5A882   /* 暖沙色，中性 */
--palette-d:  #8B7BAB   /* 柔紫，神秘 */
--palette-e:  #6B9E7E   /* 苔藓绿，自然 */
--text:       #2A2018   /* 深暖棕，极少用 */
--text2:      #8A7A6A   /* 次要文字，更少用 */
```

颜色原则：**色彩即语义**。每种颜色对应一个概念，不解释——让颜色说话。
色彩选配要和谐，像调色盘，不是数据图表。

---

## 布局原则

**色彩区块 + 极简文字**，像装置艺术标签：

```
[大色块 A]  ← 颜色本身传达概念
  一句话
  
[大色块 B]
  一句话

[色彩关系说明]  ← 最后才出现文字解释
```

- 色块面积大，文字少
- 文字是「色彩的注脚」，不是主角
- 留白是设计的一部分，不是空缺
- 允许不对称排列
- 最后有一行「色彩在说什么」的整体解读

---

## 排版

- 标题：温暖棕，轻柔，不全大写
- 色块标注：极短，2-4 字，在色块内部或旁边
- 正文：极少，只在必要时出现
- 整体感：像艺术展的说明牌，不是分析报告

---

## 特有视觉元素

- **大面积色块**：每个概念一块颜色，色块面积大于文字
- **色彩说明**：色块旁的短句，像艺术品标签
- **调色盘感**：所有颜色放在一起有整体和谐感
- **大量留白**：页面填充率 ≤ 50%
- **无图表元素**：没有箭头、没有矩阵、没有编号

---

## 收口方式

**色彩感悟**，一句话，无结论：

- 「这些颜色一起，说的是……」
- 或者直接不收口——色彩本身就是答案

---

## 不适合

- 高密度信息（那是 INTJ/ENTJ）
- 强逻辑推导（那是 INTP）
- 步骤化（那是 ISFJ）
- 饱和高对比色（破坏安静美感）
- 文字多于色彩
