# ESTP Style — 战术简报

**认知偏好**：当下行动 · 高刺激 · 实用逻辑 · 即时判断

---

## 核心美学

高对比战术简报。像一份赛前战术板——现在的局面是什么，关键变量是什么，下一步怎么打。
不是历史分析，不是长期战略，只有「现在」和「下一步」。

---

## 设计 Token

```
--bg:        #F5F5F3   /* 近白 */
--surface:   #EAEAE7   /* 浅灰 */
--text:      #1A1A18   /* 近黑 */
--text2:     #7A7A72   /* 次要 */
--border:    rgba(0,0,0,0.14)
--hot:       #D42A00   /* 高饱和红，关键变量 + 警示 */
--hot-bg:    #FFF0EE
--now:       #1A1A18   /* 近黑大字，当前局面 */
```

颜色原则：**高对比黑白 + 高饱和红**。红色标注关键变量和即时风险，不用于装饰。

---

## 布局原则

**局面 → 关键变量 → 下一步行动**，时序紧凑：

```
[当前局面 — 大字，最显眼]

关键变量：
  ⚡ [变量 A] — 影响：……
  ⚡ [变量 B] — 影响：……

现在：[当前状态]
下一步：[立即可执行的动作]
```

- 当前局面：最大字号，占页面顶部
- 关键变量：`⚡` 标注，红色高亮
- 「下一步」：粗黑字，动词开头，立即可执行
- 没有背景故事，没有「原因分析」——只有现状和行动

---

## 特有视觉元素

- **大字局面描述**：顶部最显眼，字号最大
- **⚡ 关键变量**：红色标注，每个变量后面是「影响」
- **时间感**：「现在」「下一步」「立即」等词强调即时性
- **无历史回顾**：不写「之前是……」，只写「现在是……」

---

## 收口方式

**立即行动**，强时间感：

- 「下一步：[具体动词短语]」
- 不解释，不回顾，直接给出下一个动作

---

## 不适合

- 长期战略分析（那是 ENTJ/INTJ）
- 情感叙事（那是 INFP）
- 历史依据（那是 ISTJ）
- 大量文字说明
