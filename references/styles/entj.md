# ENTJ Style — 战略简报

**认知偏好**：系统效率 · 强执行 · 结果导向 · 结论前置

**信息组织方式**：结论前置式——核心判断在最前，证据在中间，行动指令收尾；读者先看结论，再决定是否深读。

---

## 核心美学

高管简报。像一份给决策者看的战略 deck——结论在第一行，证据在后面，行动指令在最后。
没有废话，没有感情色彩，只有「现状 → 判断 → 行动」。

---

## 设计 Token

```
--bg:        #F8F8F6
--surface:   #EEEDE8   /* 暖灰，次要区块 */
--surface-b: #1E2D1E   /* 深森绿，核心结论区 */
--text:      #1A1A18
--text2:     #7A7A70
--border:    rgba(0,0,0,0.12)
--action:    #1B5E2A   /* 深森绿，行动指令 + 核心判断 */
--action-bg: #EBF3EB   /* 绿色 tint */
--compare-a: #EBF3EB   /* 对比矩阵 A 侧 */
--compare-b: #F5EBE8   /* 对比矩阵 B 侧 */
```

颜色原则：**深森绿 = 决策/行动**，仅用于：
- 核心判断节点（最显眼）
- 行动指令栏（底部）
- 对比矩阵的「推荐」侧

其余内容灰色系，通过字重和大小建立层级。

---

## 布局原则

**结论前置，证据后置**：

```
[核心判断 — 深绿底色，最大字号]
           ↓
[对比矩阵：方案 A vs 方案 B]
  指标 1: ■■■□□  vs  ■■■■□
  指标 2: ■□□□□  vs  ■■■□□
           ↓
[支撑数据 / 约束条件]
           ↓
[行动指令 — 深绿，动词开头]
```

- 第一个区块永远是结论或核心判断，不是铺垫
- 对比矩阵：两列并排，突出差异，标注「推荐」
- 行动指令区：深绿背景，白字，动词开头（「立即」「优先」「停止」）
- 不允许开放收口，不允许「也许」

---

## 排版

- 核心判断：最大字号（22px+），字重 700，深绿背景或深色强调
- 矩阵标题：小号大写，字重 600，灰色
- 行动指令：动词开头，完整句，字重 600
- 数据：数字放大，单位缩小（强调结果）
- 不允许长段落，最多 2 行说明

---

## 特有视觉元素

- **结论首栏**：整行深绿背景，白字大号，视觉权重最高
- **对比矩阵**：双列，有「推荐」标签（绿色角标）
- **进度条 / 评分**：`■■■□□` 风格，快速传达相对优劣
- **行动指令栏**：底部深绿色区块，「立即 X」「优先 Y」「停止 Z」
- **数字放大**：关键指标数字比周围文字大 1.5-2 倍

---

## 收口方式

**强行动指令**。收口是一条或多条可执行的「下一步」，动词开头，不含条件句。

ENTJ 读者不需要「这是一个有趣的问题」，他们需要**知道接下来做什么**。

---

## 不适合

- 开放收口（那是 ENTP / INFP）
- 条件句结论（那是 INTP）
- 情绪化色彩（那是 INFP）
- 步骤手册（那是 ISFJ——ENTJ 看的是战略，不是操作）
- 大量留白（浪费空间，ENTJ 需要高密度）

---

### 签名视觉元素（v2）— 必须实现的布局语法

**① 结论首栏（Verdict Bar）**
- 全宽深森绿背景（`--action: #1B5E2A`），白字
- 左侧小标注「核心判断」（10px，`rgba(white,0.6)`），下方大字结论（18px bold）
- 这是整张卡片的视觉霸主，位于最顶部，无任何留白

**② 对比矩阵（Compare Matrix）**
- 两列并排，A 侧 `--compare-a` 绿 tint，B 侧 `--compare-b` 红 tint
- 每行：指标名称 + A 列值 + B 列值
- 推荐侧右上角绿色角标「推荐」（8px，`--action` 色，`border-radius: 2px`）
- 表头背景 `--surface`，行间距宽松

**③ 行动指令行（Action Row）**
- 左侧 4px 深森绿 accent bar
- 「立即 / 优先 / 停止」动词开头（bold），一行一条指令
- 背景 `--action-bg`，多条指令之间 0.5px 分隔线

**④ 关键数字块（Metric）**
- 数字 32px bold，单位 12px `--text2`，下方上下文标注 11px `--text2`
- 多个数字横排，等宽间距
- 数字直接说明结论，不做说明性铺垫

## 与其他风格对比

| | ENTJ | INTJ | INTP |
|---|---|---|---|
| 收口 | 行动指令 | 强结论 | 条件结论 |
| 入口 | 结论先行 | 框架先行 | 命题推导 |
| 核心色 | 深森绿（行动）| 冷蓝（框架）| 暖黄（逻辑标注）|
| 密度 | 高，数字驱动 | 高，层级驱动 | 高，分支驱动 |
