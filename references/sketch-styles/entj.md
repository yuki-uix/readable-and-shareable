# ENTJ 手绘笔记风格 — 结论前置

## 核心气质
高管简报。第一行是结论，最后是行动指令。没有废话，没有感情色彩。深森绿是唯一强调色，代表「执行」和「决策」。

## 配色
- 背景：冷灰白 #F8F8F6
- 核心：深森绿 #1B5E2A（结论条 + 行动条）
- 次要：绿色 tint #EBF3EB（证据区背景）
- 对比 A：绿浅 #EBF3EB（推荐侧）
- 对比 B：红浅 #F5EBE8（对比侧）
- 文字：近黑 #1A1A18
- 次要文字：#7A7A70

## 布局骨架
自上而下，结论最重：
- 顶部：全宽深森绿结论条（最大最重）
- 中部：对比矩阵（双列）或证据区块
- 底部：行动指令条（深绿，动词开头）
- 数字单独放大（32px+）

## 线条语言
- 框架线：直线，略粗，有意图感
- 矩阵线：细直线，均匀
- 结论条/行动条：无边框，纯色块
- 行动条左侧：4px 深绿 accent bar

## 排版特征
- 结论标题：最大字号，白字，深绿底
- 矩阵标题：全大写小字，灰色
- 行动指令：动词开头（「立即」「停止」「优先」），粗体
- 数字：单独放大，单位缩小
- 全程无感情色彩词汇

## 装饰词汇
- 「推荐」角标（绿色小方块）
- 进度条 ■■■□□ 风格
- 数字大字块
- 无圆形气泡、无星星、无手绘插画

## 不得出现
- 暖色（失去决策感）
- 开放收口（ENTJ 必须给结论）
- 手写潦草感（要精准）
- 情感化词汇
