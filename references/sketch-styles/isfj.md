# ISFJ 手绘笔记风格 — 步骤手册

## 核心气质
温暖的操作手册。每步清晰编号，有做完的完成感。柔绿色是主色调，像「通关了！✅」的满足感。比 ISTJ 更温暖，更「我来帮你做到」，不是「这是规定」。

## 配色
- 背景：柔绿白 #F5FBF7
- 完成区背景：柔绿 #E8F5EC
- 完成状态绿：深绿 #2A6B3C
- 警告区：浅琥珀 #FEF3E2（注意事项）
- 警告色：暖橙 #C97C2A
- Done Bar：深绿 #2A6B3C（底部全宽）
- 文字：深暖 #1A2A1A

## 布局骨架
垂直步骤列表：
- 每步：圆圈编号 ①②③ + 步骤标题 + 说明
- 完成条件：「当……时视为完成」小字注释
- 警告块：缩进，橙色左边框（注意事项）
- 底部：Done Bar（「✅ 全部完成」深绿全宽）

## 线条语言
- 步骤连接：细虚线（向下箭头）
- 编号圆圈：手绘圆，柔绿填充
- 警告块边框：左侧 3px 橙色实线
- Done Bar：无边框，纯色块

## 排版特征
- 步骤编号：圆圈 ①，柔绿，居左
- 步骤标题：中等粗体，黑色
- 说明文字：小号，灰色
- 完成条件：斜体，更小字号
- 收口：「✅ 按步骤执行，到这一步你就完成了」

## 装饰词汇
- ✅ 完成图标（步骤旁 + Done Bar）
- ⚠️ 注意图标（警告块）
- 圆圈编号 ①②③（手绘风格）
- 步骤连接虚线箭头
- 1-2 个简单插画（对号图、小人做动作）

## 不得出现
- 冷色调（失去温暖感）
- 开放收口（ISFJ 需要明确完成）
- 放射状布局（步骤必须是线性的）
- 过于复杂的矩阵
