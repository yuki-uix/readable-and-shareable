# ENFJ 手绘笔记风格 — 成长路线图

## 核心气质
「你可以走到这里」的旅程地图。琥珀色蜿蜒路径连接编号阶段节点，每个节点有手绘小图标和注解文字，最终抵达终点。像导师给某人画的成长路线，充满方向感和温暖邀请。

参考案例：`examples/your-ai-agent-has-more-access-than/infographic/prompts/01-enfj-journey.md`

---

## 配色
- 背景：暖白
- 单一强调色：琥珀橙 #E8A020（路径线、重点节点高亮）
- 节点：白填充 + 黑手绘圆角矩形边框
- 文字：黑色
- 图标：黑色简笔画

**严格单色系**：只用琥珀作为唯一强调色。

---

## 布局骨架
S 型蜿蜒路径，从上到下：
- 顶部：大号标题
- 路径：黑色墨水蜿蜒路径，连接 3-5 个编号节点
- 每个节点：数字编号（圆圈）+ 阶段名称 + 手绘小图标 + 注解文字（节点旁）
- 节点编号用琥珀高亮
- 底部：收口洞见句 + 来源

---

## 线条语言
- 路径：黑色手绘蜿蜒曲线（有厚度）
- 节点边框：圆角矩形，手绘感
- 箭头：路径末端方向指示
- 图标：黑色简笔 2-3 笔

---

## 排版特征
- 节点编号：大号，琥珀色圆圈内白字
- 阶段名称：粗体黑色
- 注解：正文大小，节点旁排列
- 收口：大号粗体黑色问句或邀请语

---

## Prompt 关键词

```
Hand-drawn journey roadmap. Warm off-white background. Black ink winding path. Amber (#E8A020) as single accent color for nodes and highlights. Portrait ratio 2:3.
Winding S-curve path connecting {N} numbered stage nodes. Each node: numbered circle (amber highlight) + stage label + hand-drawn icon + annotation text.
```

---

## 不得出现
- 绿色终点（去掉路径渐变到绿的设定，统一用琥珀单色）
- 无编号节点（ENFJ 的节点必须有明确序号）
- 冷色调
- 强制性语气（ENFJ 是邀请，不是命令）
