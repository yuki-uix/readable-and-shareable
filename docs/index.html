How it works
四步，从文章到可分享链接
01
输入文章
粘贴文本、给文件路径、或一个 URL。支持中英文。
02
结构分析
自动识别层级型、数值型、时序型、对比型、关系型结构。
03
生成可视化
每种结构匹配对应形式。不适合可视化的，直接说明原因。
04
一键分享
上传到 sharehtml.zhenjia.dev，获得 7 天有效链接，发给任何人。
Methodology
结构类型 × 可视化形式

不是每篇文章都适合可视化。这是判断逻辑：

层级型
有章节嵌套、从属关系、"包括"/"分为"
→ 思维导图
数值型
有数字对比、成本/性能数据、优化前后量化结果
→ 图表 / 计算器
时序型
有步骤编号、实施顺序、"先做 A 再做 B"
→ 地铁图 / 流程
对比型
有"A vs B"、取舍分析、多方案评估
→ 对比表格
关系型
多个概念相互引用、系统组件依赖
→ 知识图谱
叙事型
纯散文、情感型、论证密度极高
⚠ 不建议
