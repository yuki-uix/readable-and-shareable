How it works

四步，从文章到可分享链接

01
输入文章
粘贴文本、给文件路径、或一个 URL。支持中英文。
02
结构分析
自动识别层级型、数值型、时序型、对比型、关系型结构。
03
生成可视化
每种结构匹配对应形式。不适合可视化的，直接说明原因。
04
一键分享
上传到 sharehtml.zhenjia.dev，获得 7 天有效链接，发给任何人。

Install

在 Claude Code 里使用

# 克隆 repo
git clone https://github.com/yuki-uix/readable-and-shareable.git
cd readable-and-shareable

# 安装命令（全局可用）
mkdir -p ~/.claude/commands
cp .claude/commands/viz.md ~/.claude/commands/viz.md

# 在 Claude Code 终端中触发
/viz

# 然后粘贴文章内容，或给文件路径 / URL
