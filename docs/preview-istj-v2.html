Prompt Cache 命中率低于 50% 时，必须审查前缀稳定性，不得直接新增 MCP 工具

现行
R-001
System prompt 中不得包含动态时间戳。时间信息必须移至消息末尾，以保证前缀稳定性。
来源：Prompt Cache 核心规则 · cache miss 成本分析 2026.05
现行
R-002
MCP 工具注册顺序必须固定，不允许随机化或按使用频率动态排序。
