整体洞见
优化成本的本质是管理一组矛盾：你想让上下文更稳定（缓存），又想让它更精简（压缩）。这两个目标不能同时最大化——能做的只有在两者之间找到合理的平衡点，同时警惕 Write 策略带来的语义漂移。
